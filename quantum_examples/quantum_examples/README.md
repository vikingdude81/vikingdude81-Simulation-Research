# Quantum Examples

This directory contains interactive quantum physics simulations and visualizations.

## Available Examples

### Quantum State Visualization
- **bloch_sphere.py** - Visualize qubit states on the Bloch sphere
- **hydrogen_atom.py** - Atomic orbital visualizations and quantum numbers
- **quantum_3d.py** - 3D harmonic oscillator states
- **quantum_4d.py** - 4D quantum system projections
- **random_quantum.py** - Generate and visualize random quantum states
- **schrodinger_cat.py** - Superposition states and Schrödinger's cat

### Quantum Dynamics
- **rabi_oscillations.py** - Two-level system dynamics and Rabi oscillations
- **wavepacket_evolution.py** - Time evolution of Gaussian wavepackets
- **quantum_decoherence.py** - Decoherence effects in quantum systems

### Quantum Phenomena
- **double_slit.py** - Classic double-slit interference experiment
- **quantum_tunneling.py** - Tunneling through potential barriers
- **quantum_entanglement.py** - Entangled state evolution

### Quantum Computing
- **quantum_gates.py** - Quantum gate operations (Pauli, Hadamard, CNOT, etc.)

### Comprehensive Tools
- **all_visualizations.py** - Run multiple visualizations in sequence

## Usage

Use the **Quantum Examples** workflow to run the featured example (Schrödinger's cat), or run any example directly:
```bash
python quantum_examples/bloch_sphere.py
python quantum_examples/double_slit.py
python quantum_examples/hydrogen_atom.py
# ... or any other example
```

## Output

Visualizations are saved to the `visualizations/` directory in the root of the project.
